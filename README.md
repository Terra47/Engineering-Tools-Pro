# Engineering Tools Pro

Site com calculadora científica, biblioteca de fórmulas e catálogo de pontes para estudantes e profissionais de engenharia.

## Funcionalidades
- Calculadora científica com funções trigonométricas, estatísticas e memória
- Biblioteca de fórmulas de geometria, estruturas, física, hidráulica e eletricidade
- Executor de fórmulas com cálculo automático
- Catálogo de pontes com descrições, vantagens e desvantagens

## Tecnologias
HTML, CSS, JavaScript, Bootstrap, MathJax
